# SmartCart - Customer Segmentation/Clustering (Unsupervised ML)

## Overview
Segments customers into 4 clusters based on spending behavior and demographics using KMeans clustering.

## Dataset
- 49,582 entries, 22 features
- Features: Income, Spending, Age, Education, etc.

## Tech Stack
- Python, Pandas
- Scikit-learn, Seaborn, Matplotlib
- Streamlit

## Models Used
- KMeans Clustering
- 4 Customer Segments identified

## How to Run
pip install -r requirements.txt
streamlit run app.py
